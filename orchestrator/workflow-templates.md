# Workflow Templates

> Pre-defined skill chains for common tasks.

## Evaluator-Optimizer Loop
**When:** After creating any significant output
```
express → signal-check → express (improved)
```

## Deep Research
**When:** Exploring a complex topic
```
analyze → signal-check → express (findings)
```

## Content Creation
**When:** Writing an article, post, or document
```
analyze (research) → distill (key points) → express (draft) → signal-check (review) → express (final)
```

## Book/Article Processing
**When:** Processing a long text for your knowledge base
```
distill → capture (key insights) → express (summary note)
```

## Session End
**When:** Wrapping up a productive session
```
[current work] → handoff
```

## Decision Making
**When:** Need to make a structured decision
```
analyze (options) → signal-check (evaluate) → capture (decision + reasoning)
```
